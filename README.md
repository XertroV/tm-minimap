# Trackmania Minimap

It's a minimap.

Cycle through modes: small, large, and hidden using 'M' (by default).

Features:
- Minimap Screenshot Wizard: take screenshots of maps to use as minimap backgrounds. (access via settings > race > minimap > bg image wizard)
